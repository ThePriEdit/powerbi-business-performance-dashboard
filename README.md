# 📊 Power BI Business Performance Dashboard

## Project Overview
An interactive Power BI dashboard analysing sales revenue, customer trends, 
and KPIs across 4 regions and 3 product categories — built using real transactional data 
of 214 orders across Jan–Dec 2024.

## 📸 Dashboard Preview
![Dashboard Screenshot](screenshots/dashboard.png)

## 🔍 Key Insights
- Identified **Electronics** as the top revenue segment at ~45%, followed by **Furniture** at ~29%, together contributing ~74% of total revenue
- **North region** leads revenue (₹2.8L) across all product categories
- Revenue shows a consistent upward trend across the analysis period
- Dashboard enables real-time filtering by Region and Product Category using interactive slicers

## 📌 How to View
1. Download `Business_Performance_Dashboard.pbix`
2. Open with [Power BI Desktop](https://powerbi.microsoft.com)
3. Use Region and Category slicers to filter all visuals simultaneously

## 📊 Dataset Summary
- **214 orders** across Jan–Dec 2024
- **4 Regions:** North, South, East, West
- **3 Product Categories:** Electronics, Furniture, Clothing
- **Metrics tracked:** Revenue, Cost, Profit per order
